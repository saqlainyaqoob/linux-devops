# Day 2 – Files, Permissions & Users

## Permissions
- rwx model (owner, group, others)
- Numeric permissions (755, 644)

## Commands Used
chmod, chown, ls -l, useradd, groupadd, usermod

## Bash Script
- Script accepts filename as argument
- Checks existence and prints permissions

## PRACTICED
*  chmod → permission control ✔ 
*  chown → ownership control ✔
*  useradd, groupadd, usermod → access management ✔ 
*  Bash arguments ($1) ✔ 
*  File existence checks ✔
